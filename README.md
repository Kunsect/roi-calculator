# Keywords ROI Calculator - 导入 Semrush Excel 自动计算关键词优化回报率

在 Semrush 搜索关键词并导出 xlsx 之后，可以导入到该项目，会通过公式自动计算每个关键词对应的优化回报率。Excel 均在本地设备处理，不会上传到任何远程服务器。
